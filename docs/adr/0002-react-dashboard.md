# ADR 0002: Escolha do frontend

## Status
Accepted

## Context
É necessário uma interface para acompanhar scans e findings com atualização simples e baixa curva de aprendizado.

## Decision
Usar React 19 + TypeScript + Vite para uma interface reativa e moderna.

## Consequences
- Boa experiência de desenvolvimento local.
- Facilidade de integração com a API REST.
- Caminho simples para evoluir para dashboards mais ricos.
